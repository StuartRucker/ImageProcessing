What is iStitch

iStitch is a image processing system that calculates features with-in a cornea. It first stitches the images into one by using the opencv stitching algorithm. Then, it applies multiple filters to find just the veins within the image, and calculates statistics of those veins. For example, it finds corneal nerve fiber tortuosity (CNFTo), corneal nerve branch density (CNBD), corneal nerve fiber density (CNFD), and corneal nerve fiber length (CNFL).
